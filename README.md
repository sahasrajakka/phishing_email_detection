# Phishing Email Detection Model

A machine learning-based system that identifies and classifies phishing emails by analyzing email content, suspicious links, and sender characteristics. The project helps users detect fraudulent emails and improve awareness of cybersecurity threats.

## Features
- Email classification (Phishing / Legitimate)
- Suspicious keyword detection
- URL analysis
- Machine Learning-based prediction
- Security risk alerts

## Technologies Used
- Python
- Scikit-learn
- Pandas
- NumPy
- Natural Language Processing (NLP)

## How to Run

```bash
python phishing_detector.py
